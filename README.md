# The Sand Fountain Hotel – AR Project

This project is part of the **Extending Digital Narrative (XDN)** research group, led by **Scott Rettberg**, from the **Center Digital Narrative** (Bergen, Norway).

**Grand Hotels** scenarios are written by **Roderick Coover**, **Caitlin Fisher**, and **Scott Rettberg**.  
It is inspired by Robert Coover’s book *The Grand Hotels (of Joseph Cornell)*, and explores how literature and art can be reimagined through immersive technologies.

This is the repository for the Sand Fountain Hotel.

---

## Project Overview
- A series of **AR/VR “Grand Hotel” experiences**, each based on Coover’s poetic vignettes.  
- Developed using **Unity** and AR Foundation Framework.  
- Combines **narrative, interactivity, and immersive design** to create new ways of experiencing storytelling.  
- Includes a **central lobby application** that gives access to the different hotels.  

---

## Technology
- Built primarily with **Unity 3D Engine**.  
- Uses **Git LFS** to manage large media assets (3D models, textures, sound).  

---

## Inspiration
The project is inspired by **Robert Coover’s** *The Grand Hotels (of Joseph Cornell)* (2002), which itself was a literary response to the surreal box assemblages of **artist Joseph Cornell**.
